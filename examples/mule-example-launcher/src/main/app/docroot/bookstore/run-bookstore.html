Running the Example:

The Bookstore example is configured to run in an embedded jetty servlet container right from the Mule server. The applications will then be available at the following URLs:

  • Main Bookstore App (search for and order books): http://localhost:8083/bookstore
  • Admin App (add books and get order stats): http://localhost:8083/bookstore-admin
